LeetCode 7 - Reverse Integer

LeetCode 7 - Reverse Integer

👉 題目連結

📖題目說明

Given a signed 32-bit integer x, return x with its digits reversed.
If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0.
Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

給定一個 32 位元帶號整數 x，將它的數字反轉後回傳
如果反轉後超出範圍 [-2^31, 2^31 - 1]，就回傳 0
不允許使用超過 64 位的整數存放。

範例輸入與輸出

Example 1

Input: x = 123
Output: 321

Example 2

Input: x = -123
Output: -321

Example 3

Input: x = 120
Output: 21

Constraints

  • -231 <= x <= 231 - 1

Solution

字串處理

解題想法:
直接把數字轉成字串處理，因為這樣反轉會非常直觀

  1. 先判斷 x 是否為負數
  2. 把數字轉成字串並反轉
  3. 再轉回整數，補上正負號
  4. 檢查是否超出 32 位範圍，若超出就回傳 0
    Python 本身的整數沒有 32 位溢位問題，可以直接處理大整數
    時間複雜度: O(n)，n表示數字的位數，主要花費在 str(x) 和 [::-1] 的操作
    空間複雜度: O(n)，因為要建立字串 str(x) 和反轉後的新字串

數學pop & push

解題想法: 改用數學方式，逐位取出數字並組合回去

  1. 先記錄數字的正負號。
  2. 每次用 %10 取出最後一位數（pop）
  3. 將結果往左移一位（乘 10）並加上這個數字（push）
  4. 過程中檢查是否會超過 2^31-1 或小於 -2^31
  5. 最後補回正負號
    這種方法比較貼近題目的考點，比用字串反轉的效率更高且更省空間
    時間複雜度: O(n) ,n 表示數字的位數
    空間複雜度: O(1) ,只用了常數變數
