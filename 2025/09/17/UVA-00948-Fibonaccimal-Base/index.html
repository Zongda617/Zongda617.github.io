UVA 00948 - Fibonaccimal Base

UVA 00948 - Fibonaccimal Base

👉 題目連結

📖題目說明

這題要求把一個整數轉換成斐波那契進位制，即用斐波那契數列來表示數字
任何正整數都可以用不連續的斐波那契數唯一表示

輸入與輸出

輸入

輸入的第一行含有一個數字 N，代表以下有幾個數字 ( 1 ≤ N ≤ 500)
接下來有 N 行，每行有一個小於 100000000 的正整數

輸出

每一個整數需輸出一行，格式為: “DEC_BASE = FIB_BASE (fib)”
DEC_BASE 原始的十進位數字而 FIB_BASE 則是它的費氏進位表示法

⚠️注意事項與陷阱

  1. 表示必須唯一: 不能出現兩個連續的斐波那契數字
  2. 輸出格式: 中間要有 “ = “，後面要加上 “ (fib)”

解題想法

運用貪婪演算法
先生成 ≤10^9 的 Fibonacci
從最大開始往下掃，能取就輸出 1 並減去，否則輸出 0
去掉前導零就是答案
