UVA 10008 - What's Cryptanalysis

UVA 10008 - What’s Cryptanalysis

👉 題目連結

📖題目說明

輸入一段包含多行文字的訊息，計算所有英文字母的出現次數

輸入與輸出

輸入

第1列有一個正整數N，代表以下有多少列文字
接下來的N列，每行是一段文字，可能包含大小寫英文字、空格、標點符號、數字

輸出

每一行輸出一個字母與它的出現次數
僅輸出出現過的字母，出現次數不能為0
排序規則

  1. 出現次數由大到小
  2. 如果次數相同，則依 字母 A–Z 升序排列。

⚠️注意事項與陷阱

  1. 大小寫要統一: 字母不分大小寫，必須先轉成大寫再統計
  2. 非字母忽略: 只計算 A–Z，數字、標點、空格都要跳過
  3. 只輸出有效字母: 出現次數為 0 的字母不可列印
  4. 輸出格式: 每行必須是「字母<空格>次數」

解題想法

使用 Hash Table，將字母作為 key，出現次數作為 value
讀入文字後，先判斷字元是否為英文字母，如果是就轉成大寫並累加次數，其他符號則直接忽略
最後將統計結果依照「出現次數由大到小」排序，若次數相同則依字母順序排列，輸出時只列出有出現的字母即可
