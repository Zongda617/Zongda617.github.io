Markdown 是什麼？

Markdown 是一種 輕量級標記語言（Lightweight Markup Language），主要用來快速編寫排版良好的文件
它的特色是：

  • 使用簡單的文字符號（如 #、*、> 等）來控制格式。
  • 可輸出成 HTML，並被廣泛應用在 GitHub、部落格、筆記（如 Notion、Obsidian）、文件編寫等場合。*
  • 讓人能專注於內容，而不是複雜的排版
