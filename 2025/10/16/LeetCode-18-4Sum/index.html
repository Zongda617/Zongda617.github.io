LeetCode 18 - 4Sum

LeetCode 18 - 4Sum

👉 題目連結

📖題目說明

Given an array nums of n integers, return an array of all the unique quadruplets [nums[a], nums[b], nums[c], nums[d]] such that:

  • 0 <= a, b, c, d < n
  • a, b, c, and d are distinct.
  • nums[a] + nums[b] + nums[c] + nums[d] == target
    You may return the answer in any order.

給一個數組nums，傳回一個由唯一四元組[nums[a], nums[b], nums[c], nums[d]]組成的數組，使得：

  • 0 <= a, b, c, d < n
  • a、b、c 和 d 是不同的
  • nums[a] + nums[b] + nums[c] + nums[d] == target
    您可以以任意順序返回答案

範例輸入與輸出

Example 1

Input: Input: nums = [1,0,-1,0,-2,2], target = 0
Output: Output: [[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]

Example 2

Input: nums = [2,2,2,2,2], target = 8
Output: [[2,2,2,2]]

Constraints

  • 1 <= nums.length <= 200
  • -10^9 <= target <= 10^9
  • -10^9 <= nums[i] <= 10^9

Solution

排序 + 雙指針 + 剪枝

解題想法:
這題我採用排序 + 雙指針的方式，算是經典的 k-Sum 模板延伸到 4Sum
先固定前兩個數，再用雙指針在剩下的區間中尋找另外兩個數
過程中要特別注意重複元素與剪枝條件，否則時間會炸開或產生重複解

  1. 先將陣列排序，方便去重與指針移動
  2. 接著固定前兩個數 nums[i]、nums[j]
  3. 再用雙指針在右側區間中搜尋剩下兩個數，使四數總和等於 target
  4. 為提升效率，加入上下界剪枝：
    • 如果最小四數和已大於 target，提前結束迴圈
    • 如果最大四數和仍小於 target，直接略過
  5. 雙指針部分依據總和大小調整左右指標，並在找到組合後跳過重複值，避免重複解

時間複雜度: O(n³)，n 是陣列的長度，外層雙層迴圈搭配內層線性雙指針搜尋
空間複雜度: O(1)，僅使用常數額外變數
