LeetCode 24 - Swap Nodes in Pairs

LeetCode 24 - Swap Nodes in Pairs

👉 題目連結

📖題目說明

Given a linked list, swap every two adjacent nodes and return its head.
You must solve the problem without modifying the values in the list’s nodes (i.e., only nodes themselves may be changed.)

給你一個鏈結串列，請你兩兩交換其中相鄰的節點，並返回交換後鏈結串列的頭節點
限制：你必須在不修改節點內部值的情況下完成

範例輸入與輸出

Example 1

Input: head = [1,2,3,4]
Output: [2,1,4,3]

Example 2

Input: head = []
Output: []

Example 3

Input: head = [1]
Output: [1]

Example 4

Input: head = [1,2,3]
Output: [2,1,3]

Constraints

  • 串列中的節點數量在 [0, 100] 範圍內
  • 0 <= Node.val <= 100

Solution

迭帶法

解題想法:
先設定了一個 dummy 虛擬節點，方便處理節點交換
我使用一個 current 指標，它始終指向待交換那對節點的前一個節點
接著交換兩個節點並跟新current指標直到串列結束
最後返回 dummy.next 即為新頭節點
時間複雜度: O(N)，N是串列的節點數
空間複雜度: O(1)，只使用了幾個額外的指標

遞迴法

解題想法:
將問題拆解成交換當前這一對和交換剩下的所有對
swapPairs函式被賦予一個任務：交換 head和 head.next，然後返回交換後的新頭節點
如果串列為空，或者只剩下一個節點，顯然無法交換，直接返回 head

時間複雜度: O(N)，N是串列的節點數
空間複雜度: O(N)，主要為遞迴呼叫的堆疊深度
