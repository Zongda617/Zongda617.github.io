LeetCode 15 - 3Sum

LeetCode LeetCode 15 - 3Sum

👉 題目連結

📖題目說明

Given an integer array nums, return all the triplets
[nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.
Notice that the solution set must not contain duplicate triplets.

給定一個整數數組 nums，傳回所有三元組 [nums[i]、nums[j]、nums[k]]
使得 i != j、i != k 和 j != k，並且 nums[i] + nums[j] + nums[k] == 0
解決方案集不得包含重複的三元組

範例輸入與輸出

Example 1

Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
Explanation:
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0.
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
The distinct triplets are [-1,0,1] and [-1,-1,2].
Notice that the order of the output and the order of the triplets does not matter.

Example 2

Input: nums = [0,1,1]
Output: []
Explanation: The only possible triplet does not sum up to 0.

Example 3

Input: nums = [0,0,0]
Output: [[0,0,0]]
Explanation: The only possible triplet sums up to 0.

Constraints

  • 3 <= nums.length <= 3000
  • -10^5 <= nums[i] <= 10^5

Solution

排序 + 雙指針

解題想法:
將3Sum 降維成 2Sum

  1. 先對數組進行排序，方便去除重複數字以及搜尋
  2. 遍歷每個數字作為固定的第一個數字 a，然後在其右側子陣列中使用左右指針，尋找兩個數字 b、c 使得 a + b + c = 0
  3. 移動指針：
    若三數和小於 0，左指針右移（讓總和變大）
    若三數和大於 0，右指針左移（讓總和變小）
    若剛好為 0，記錄結果並同時內縮兩指針
  4. 為避免重複組合，當固定數字或左右指針的值與前一個相同時，直接略過
  5. 若固定的數字大於 0，因為排序後右邊也都是非負數，不可能再湊成 0，可直接停止搜尋
    時間複雜度: O(n²)，n 是陣列的長度，因為每個固定數字都搭配一次線性雙指針搜尋
    空間複雜度: O(1)，只使用常數級的指標變數

Hash Set

解題想法:
我先鎖定一個數，把問題簡化成 Two Sum，用 Hash Set 當成快速查表

  1. 先對數組進行升序排序，讓相同數字集中，方便後續統一處理與結果輸出
  2. 逐一固定每個數字 a，並設定目標值 target = -a，接著在其右側子陣列中尋找兩個數字 b、c 使得 b + c = target
  3. 建立一個 seen 集合，當遍歷到每個 b = nums[j] 時，檢查 target - b 是否已存在於 seen
  4. 每一次遍歷都將 b 加入 seen 以便之後比對
    因為結果使用 set 儲存三元組，即使同樣的數組合出現多次，也只會保留一份
    時間複雜度: O(n²)，n 是陣列的長度，外層固定一個數、內層利用 Hash 掃描整個子陣列。
    空間複雜度: O(n)，每次固定數字時使用的 Hash Set 可能最多存放 n 個元素。
